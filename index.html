    About Motus

    Motus is a web App that can carry out expression analysis by taking real time video input and then effectively describe the emotion.

    Motive

    The project can serve as a learning platform through which the people can visually learn to recognise emotions in real time.

    Who are the target?

    Alexithymia is a broad term to describe problems with feeling emotions. In fact, this Greek term used in Freudian psychodynamic theories loosely translates to “no words for emotion.” While the condition is not well-known, it’s estimated that 1 in 10 people has it.

    Technologies used

    HTML, CSS and JavaScript, Flask Framework, OpenCV for the necessary image processing

    Frequently Asked Questions

    • How to use the application?

      Click on Open Application from the menu bar. Allow access to webcam and make sure you are in a well lit surrounding.

    • How does the model work?

      The model is a convolutional neural network that has been trained. It contains 4 conv2D layers that can perform feature extraction and can predict 7 emotions.

    • How was openCV used?

      OpenCV has been used for the necesary image processing and resampling of over 28000 images in the training set. OpenCV is also used for the Face Detection and taking live video input from the user.

    • What processes were used for the dataset resampling?

      Images were resampled with ImageDataGenerator from keras. The images underwent rescaling(dividing each pixel by 255), Zoom change(range=0.2), Shear Change(range=0.2), horizontal flip(true) and rotation(range=0.2).
